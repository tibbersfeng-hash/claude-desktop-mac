## 一、项目背景

Claude Desktop Mac 是一个原生 macOS 桌面应用，核心定位是**替代 Claude Code CLI**，为本地 Claude Code 提供图形化前端界面。

根据 ROADMAP.md，项目已标记为"开发完成"，包含 6 个阶段。

### 5.2 用户核心需求

**必须实现 (P0)**:
1. 消息编辑/重新生成 - 用户场景：修正输入错误或重新生成AI回复
2. MCP Server 管理 - 用户场景：配置和扩展 Claude 能力
3. 多项目管理 - 用户场景：同时工作多个项目，快速切换

**重要功能 (P1)**:
1. 快捷键自定义 - 可选功能，基本快捷键足够
2. 终端集成 - 可选功能，查看命令输出
3. 性能优化 - 用户非常重视启动速度和响应时间

**可选功能 (P2)**:
1. 会话导出 - 备份和迁移场景
2. 会话分组/标签 - 大量会话管理

**不需要的功能**:
1. 代码补全 (Tab) - CLI 对话模式足够
2. 文件树浏览 - CLI 文件操作足够
3. 协作功能 - 个人使用为主
