## 一、会话管理 (Session Management)

### 功能名称: 会话列表

**分类**: 核心功能

**优先级**: P0

**必要性评分**: 10

**UI 设计**:
- 界面位置：左侧侧边栏，220px 宽
- 交互元素：会话项列表，支持点击选择
- 视觉样式：48px 高度，图标 + 标题 + 时间
- 动效要求：悬停高亮，选中状态背景色

**实现流程**:
1. 加载本地存储的会话数据
2. 按更新时间倒序排列
3. 渲染 LazyVStack 列表
4. 绑定点击事件切换会话

**预期效果**:
- 用户看到所有历史会话
- 点击即可切换到对应会话
- 系统响应时间 < 100ms

**技术要点**:
- 涉及模块：SessionViewModel, SessionStore
- 依赖关系：本地存储 (FileManager)
- 难点分析：大量会话时的虚拟滚动

**参考实现**:
- Claude Code CLI: 会话持久化机制
- ChatGPT Mac: 侧边栏会话列表设计
